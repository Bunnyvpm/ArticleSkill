# 公众号爆款写作 Skill 系统

> **通用版** - 让AI帮你写出爆款文章的完整写作系统

一个强大、灵活、易用的AI写作助手，适用于各类创作者。系统会学习你的写作风格，适配你的人设，帮你创作出高质量的公众号文章。

---

## ✨ 核心特性

### 🎯 风格自适应
- 自动学习你的写作风格
- 支持多种人设类型（亲和力强/专业严谨/幽默风趣/极简干货）
- 根据你的配置调整文章风格

### 📊 智能热点抓取
- 自动抓取相关热点话题
- 分析爆款文章切入角度
- 提供可借鉴的写作要素

### 📝 完整写作流程
- 从选题到发布的全流程支持
- 4种标题公式，3种文章结构
- 爆款要素自动检查

### ✨ AI文本人工化
- 去除AI高频词汇和机械连接词
- 增加人性化细节和场景化描述
- 让AI生成的文本更像真人手笔

## 🎨 支持的风格

### 亲和力强型
```
特点：第一人称、情绪化表达、适度自嘲
适合：个人分享、经验教程
示例：
"说实话，用这个工具的时候我心态都崩了..."
"连我这种菜鸡都能学会，你也可以"
```

### 专业严谨型
```
特点：数据支撑、客观分析、逻辑严密
适合：行业分析、产品评测
示例：
"根据测试数据，性能提升了50%..."
"从产品角度分析，这个设计..."
```

### 幽默风趣型
```
特点：轻松调侃、生动比喻、口语化
适合：轻松教程、生活分享
示例：
"这玩意儿...怎么说呢，用起来直呼卧槽"
"就像给你的大脑装了个加速器"
```

### 极简干货型
```
特点：话少干货多、直击要害、少修饰
适合：技术文档、快速教程
示例：
"第一步：配置环境
第二步：安装依赖
第三步：运行代码"
```

## 🔧 核心功能

### 1. 标题生成器
提供4种爆款标题类型：
- 冲突对比型："我都不知道自己写了什么代码，但它在Github上火了"
- 疑问引导型："AI工具，到底是一个均衡器，还是一个放大器？"
- 数字+效果型："600星背后：我的AI工具开发日记"
- 否定+反转型："别再用关键词试错了！MJ风格库+NanoBanana"

### 2. 结构生成器
根据文章类型自动生成：
- 完整文章大纲
- 每部分字数建议
- 配图位置提示
- 写作要点提醒

### 3. 内容优化器
优化文章的：
- 语言风格（适配你的人设）
- 段落节奏（每段≤5行）
- 情绪表达（根据风格）
- 金句和互动点

### 4. 去AI化处理
去除AI生成文本的：
- 机械连接词（首先/其次/最后）
- AI高频词汇（值得注意的是/毫无疑问）
- 模板化结构
- 增加人性化细节

### 5. 爆款检查
发布前自动检查：
- ✅ 标题吸引力
- ✅ 开头抓人程度
- ✅ 内容实用性
- ✅ 互动设计
- ✅ 金句易传播性

## ⚠️ 核心原则

### 减少条目式输出

**这是最重要的写作原则**。

❌ **避免**：
```
1. 首先，你要做A
2. 然后，你要做B
3. 最后，你要做C
```

✅ **采用**：
```
做这件事的时候，大多数人会直接从A开始。但我的建议是，先花点时间
想清楚B，然后再回头做A。为什么？因为[原因]。至于C，那是水到渠成
的事，不用急。
```
