## 📝 4种爆款标题公式

### 1. 冲突对比型
```
[弱者身份] + [意外成就]
"我都不知道自己写了什么代码，但它在Github上火了"
```

### 2. 疑问引导型
```
[痛点场景] + [核心问题]
"AI工具，到底是一个均衡器，还是一个放大器？"
```

### 3. 数字+效果型
```
[数字/时间] + [结果]
"600星背后：我的AI工具开发日记"
```

### 4. 否定+反转型
```
别再[错误行为]了！[新方法]来啦
"别再用关键词试错了！MJ风格库+NanoBanana"
```


## 🔧 AI高频词黑名单

**去AI化时避免使用**:
```
连接词类：
首先、其次、最后、总之、综上所述、因此、然而

填充词类：
值得注意的是、应该注意的是、毫无疑问、一般来说、显而易见

结构词类：
一方面、另一方面、除此之外、再者

其他：
进行、实现、采用、具有（动词名词化）
```

**替换建议**:
```
"首先...其次...最后" → 用段落自然过渡
"综上所述" → 直接总结，不用连接词
"值得注意的是" → 直接说重点
"毫无疑问" → 删除，或用"确实"
```
