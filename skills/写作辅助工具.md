# 写作辅助工具

> 快速生成标题、优化内容、检查爆款要素

## 命令列表

### /标题生成
**用法**：`/标题生成 [主题/内容简述]`

**功能**：
- 生成4种类型的爆款标题（冲突对比型、疑问引导型、数字效果型、否定反转型）
- 每种类型生成3-5个选项
- 自动检查标题质量

**示例**：
```
用户：/标题生成 我用AI做了一个提示词管理工具

AI助手：
【冲突对比型】
1. "我都不知道自己写了什么代码，但我的AI工具在GitHub火了"
2. "连Hello World都写不利索的我，竟然做出了一个提示词管理神器"
3. "一个设计零基础的人，如何用AI做出600星的开源项目"

【疑问引导型】
1. "设计师的提示词管理困境，到底该怎么解决？"
2. "AI辅助开发：是均衡器还是放大器？"
3. "不会代码的设计师，能做出好用的AI工具吗？"

...（继续输出其他类型）
```

---

### /结构生成
**用法**：`/结构生成 [文章类型] [主题]`

**文章类型**：
- 工具介绍型
- 方法教程型
- 深度思考型

**功能**：
- 生成完整文章大纲
- 标注每部分字数
- 提示配图位置
- 给出写作要点

**示例**：
```
用户：/结构生成 工具介绍型 PromptFill 0.7版本更新

AI助手：
【文章结构：工具介绍型】

1. 开头钩子（150字）
   - 这周发生了一件魔幻的事：PromptFill突破600星
   - 我，一个连代码都写不利索的设计师...

2. 痛点场景（350字）
   - 先问大家一个问题：你们平时怎么管理提示词？
   - 内心独白："每次要用都要去历史记录翻，心态崩了"
   - 配图：混乱的提示词管理现状截图

...（继续完整大纲）
```

---

### /内容优化
**用法**：`/内容优化 [文章段落/全文]`

**功能**：
- 调整语言风格（符合作者配置中的人设）
- 优化段落节奏（控制每段行数）
- 添加金句和引用
- 设计互动点
- 标注配图建议

**示例**：
```
用户：/内容优化 PromptFill可以帮助用户管理提示词，支持模板和变量

AI助手：
【优化建议】

原文：PromptFill可以帮助用户管理提示词，支持模板和变量

优化后（根据作者风格调整）：
"说实话，以前我管理提示词全靠备忘录，每次要用的时候翻得心态崩了。
直到我用PromptFill做了一个模板，那种感觉——极其解压！💫
变量替换就像魔法一样，填几个空，复杂的提示词就生成了。"

优化要点：
✅ 增加"我"的第一人称视角
✅ 添加情绪化表达"心态崩了""极其解压"
✅ 使用比喻"像魔法一样"
✅ 添加emoji增强亲和力

注意：以上优化基于[亲和力强]的人设标签，如果是[专业严谨]型作者，优化方式会不同。
```

### /金句生成
**用法**：`/金句生成 [主题/观点]`

**功能**：
- 基于你的观点生成易传播的金句
- 遵循短小精悍、有对比、能共鸣的原则
- 每次生成5-10个选项

**示例**：
```
用户：/金句生成 不会代码也能用AI做工具

AI助手：
【金句选项】

1. "不懂技术不再是你无法落地的借口"
2. "我负责'想'，AI负责'写'"
3. "连代码都不会的我，都能做出AI工具，你也可以"
4. "AI时代，想法比代码更重要"
5. "技术门槛已经消失，剩下的只有你的想象力"

【推荐理由】
选项1：有对比（不懂技术 vs 落地），适合截图
选项2：对比强烈，简洁有力
选项5：消除焦虑，赋能读者
```

### /爆款检查
**用法**：`/爆款检查 [文章标题/内容]`

**功能**：
- 对照爆款要素清单逐项检查
- 标注优秀项和改进项
- 给出具体优化建议
- 估算爆款潜力

**示例**：
```
用户：/爆款检查 "我用AI做了一个提示词管理工具"

AI助手：
【爆款要素检查】

标题：我用AI做了一个提示词管理工具

检查结果：
❌ 冲突/对比：缺少反差感
❌ 具体数字：没有数字
❌ 情绪触发：平淡描述
❌ 长度：17字 ✅（适中）

【优化建议】
建议改为：
✅ "我都不知道自己写了什么代码，但它在Github上火了"
✅ "600星背后：我'不会代码'却做成了AI工具"
✅ "一个周末，我用AI做出了改变创作习惯的工具"

【爆款潜力】⭐⭐⭐（中等）
优化后可提升至：⭐⭐⭐⭐⭐
```

```
配图：流程图 + 步骤截图

...（继续更多话题）
```

### /风格适配
**用法**：`/风格适配 [文本内容]`

**功能**：
- 根据作者配置.md中的人设调整文字风格
- 自动匹配情绪化程度、口语化程度
- 调整专业术语使用频率

**示例**：
```
用户：/风格适配 这个工具使用了最新的AI技术

AI助手：
【风格适配结果】

原文：这个工具使用了最新的AI技术

根据[亲和力强]风格适配：
"说实话，这个用的AI技术还挺新的，用起来...怎么说呢，挺惊艳的"

根据[专业严谨]风格适配：
"该工具采用最新的AI技术架构，在性能和准确性上都有显著提升"

根据[幽默风趣]风格适配：
"这工具用的AI技术...怎么说呢，就是那种让你直呼'卧槽'的新"
```

## 组合使用建议

### 新手写作流程
```
1. /话题推荐    → 找到合适话题
2. /标题生成    → 生成吸引人的标题
3. /结构生成    → 搭建文章大纲
4. [撰写内容]   → 按大纲写作
5. /去AI化      → 去除AI味，让文本更自然
6. /内容优化    → 优化语言和节奏
7. /金句生成    → 添加易传播金句
8. /爆款检查    → 发布前最终检查
```

### 快速优化流程
```
已有文章 →
1. /爆款检查（检查问题）
2. /去AI化（去除AI味）
3. /内容优化（优化内容）
4. /金句生成（添加亮点）
```

### AI润色流程
```
AI生成初稿 →
1. /去AI化（去除机械连接词和AI高频词）
2. /风格适配（匹配作者风格）
3. /内容优化（增加亲和力和情绪）
4. /金句生成（添加亮点金句）
```

## 注意事项

### 风格一致性
所有优化和生成都会参考作者配置.md，确保符合作者风格。

### 灵活性
命令不必严格遵循格式，自然对话也可以，AI会理解意图。

### 迭代优化
如果第一次优化不满意，可以继续调整，AI会根据反馈持续优化。
