# 热点分析工具

> 自动化抓取热点话题，分析爆款文章结构

## 使用方式

### 方式1：直接对话
直接告诉我：
- "帮我找一下AI工具相关的热点"
- "分析一下最近设计领域的爆款文章"
- "搜索MidJourney相关的热门话题"

我会使用网络搜索工具为你获取最新信息。

### 方式2：结构化分析
使用以下命令格式：

```
/热点分析 [领域/关键词]
```

## 分析维度

### 1. 热点话题提取
- 最新的工具发布/更新
- 行业热点事件
- 流行趋势话题
- 用户讨论热点

### 2. 爆款标题分析
对于找到的热门文章，分析：
- 标题类型（冲突型/疑问型/数字型/否定型）
- 字数统计
- 关键词提取
- 情绪触发点

### 3. 文章结构拆解
分析爆款文章的：
- 开头钩子方式
- 段落结构
- 配图策略
- 互动设计

### 4. 借鉴角度建议
基于热点分析，提供：
- 可切入的写作角度
- 差异化观点
- 适合你的选题方向

## 示例对话

**用户**：
帮我分析一下AI设计工具的热点

**AI助手**：
我会为你：
1. 搜索最新的AI设计工具发布信息
2. 查找相关的爆款文章
3. 分析标题和内容结构
4. 给你可借鉴的写作角度

## 注意事项

### 时效性
热点内容变化快，建议在创作前1-2小时内进行搜索，确保信息新鲜。

### 多样性
不要只看单一来源的热点，建议查看多个平台和角度。

### 相关性
抓取热点时要结合作者配置.md中的领域和定位，不要偏离。

### 差异化
热点分析不是为了模仿，而是为了找到差异化切入点。避免跟风写雷同内容。
