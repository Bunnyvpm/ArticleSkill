# 公众号写作Skill系统说明

> **通用版** - 适用于各类创作者的AI写作助手系统

## 🎯 系统目标

当你说"我要写一篇关于XX的文章"时，AI助手会：

1. ✅ **同时读取所有Skill文件** - 确保不遗漏任何写作方法论
2. ✅ **学习你的写作风格** - 从作者配置.md和现有文章中提取风格特征
3. ✅ **抓取相关热点** - 确保话题具有时效性和爆款潜力
4. ✅ **综合所有信息** - 按照完整流程创作高质量文章
5. ✅ **适配你的风格** - 根据你的配置调整文章风格

```
├─ 如果存在 → 读取并提取人设特征

├─ 读取：公众号爆款写作.md
├─ 读取：深度认知写作.md

├─ 检查：人设一致性（对照作者配置）
├─ 检查：风格一致性
├─ 检查：爆款要素齐全
└─ 输出：最终文章
```

## 🎨 风格学习机制

AI会从作者配置.md和你的文章中学习：

### 1. 从作者配置.md学习
```
基本信息：
- 笔名、身份、领域、背景

人设标签：
- 亲和力强 / 专业严谨 / 幽默风趣 / 极简干货
- 真实感 / 数据支撑 / 情绪化表达

内容基因：
- 痛点驱动 / 产品思维 / 专业视角 / 迭代思维

价值主张：
- 实用主义 / 审美在线 / 不制造焦虑 / 深度思考

情绪表达：
- 常用情绪词汇
- 自嘲风格
- 互动习惯
```

### 2. 从现有文章学习
```
开头钩子方式：
- "这周发生了一件..."
- "先问大家一个问题"
- 引用式金句

段落长度习惯：
- 每段3-5行
- 不写长段落
- 频繁分段

情绪表达方式：
- "心态崩了"、"极其解压"、"魔幻"
- 或更加内敛专业

金句位置：
- 每个主要部分结尾
- 开头或结尾
- 引用格式强调

互动引导：
- "先问大家一个问题"
- "欢迎评论区返图"
- "如果感兴趣，欢迎来提建议"
```

## 💬 常见问题

### Q: 如果我没有填写作者配置会怎样？
A: AI会引导你填写核心信息，或使用通用默认配置。但建议填写配置以获得更个性化的效果。

### Q: 可以随时更改作者配置吗？
A: 可以。随时更新作者配置.md，AI会在下次创作时使用新配置。

### Q: 我可以同时拥有多种风格吗？
A: 可以在作者配置中组合多个标签，AI会综合考虑。例如"专业严谨+幽默风趣"。

### Q: 需要提供多少篇参考文章？
A: 建议1-3篇，选择最能代表你风格的文章。

### Q: 系统会学习我的新文章吗？
A: 会。你可以随时告诉AI学习你的新文章，它会持续优化风格理解。

### Q: 支持英文写作吗？
A: 目前主要针对中文写作优化。英文写作可以使用部分方法论，但效果可能不如中文。
