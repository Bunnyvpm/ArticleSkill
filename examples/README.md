# 示例文章目录

此目录用于存放示例文章，供 AI 学习你的写作风格。

## 如何使用

1. 在此目录下放置 1-3 篇你最喜欢的文章
2. 在 `作者配置.md` 的"参考文章"部分填写路径
3. AI 在创作时会参考这些文章学习你的风格

## 示例文件结构

```
examples/
├── README.md                # 本文件
├── tool-review-example.md   # 工具评测类文章示例
├── tutorial-example.md      # 教程类文章示例
└── thinking-example.md      # 深度思考类文章示例
```

## 注意事项

- 选择最能代表你风格的文章
- 建议选择不同类型的文章（工具介绍、教程、深度思考等）
- 文章应该是完整的，包含标题、正文等
- 不要放置其他作者的版权文章

## 示例

以下是一个示例引用（在 `作者配置.md` 中）：

```
文章1标题：600星背后：我的AI工具开发日记
文章1路径：examples/tool-review-example.md
代表风格：工具介绍、亲和力强
```

## .gitignore 说明

此目录下的实际示例文件已在 `.gitignore` 中排除，不会提交到 Git 仓库，避免包含你的私人文章内容。
